GitHub Readme Stats

This is a self-hosted instance. It serves stats cards from the /api endpoints, with private and organization contributions included.

  • /api?username=YOUR_NAME — stats card
  • /api/top-langs?username=YOUR_NAME — top languages
  • /api/pin?username=YOUR_NAME&repo=REPO — repo pin
  • /api/gist?id=GIST_ID — gist pin
  • /api/wakatime?username=YOUR_NAME — wakatime

See the project documentation for all options.
